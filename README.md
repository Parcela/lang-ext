JavaScript Language extensions
========

Provides a series of extensions to JavaScript native methods for basic object and function manipulation.  

* Object instance methods:
	* each
	* isEmpty
	* keys
	* map
	* merge
	* shallowClone
	* some
	* values
	
* Object static methods: 
	* createClass
	* merge
	
* Function
	* mergePrototypes
	* rbind
	* subClass
